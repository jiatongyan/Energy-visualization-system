# Energy-visualization-system
一个基于行业和资源的碳排放可视化系统

系统采用时间线、热力图、柱状图、饼状图等多种形式展示了碳排放总量的变化趋势，碳排放在不同行业和不同资源领域的分布情况

创新点：不同于传统的碳排放可视化系统，本系统首次将产生碳排放的行业和资源联系起来，采用热力图的形式直观展现了碳排放的来源
